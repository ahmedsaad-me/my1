How to use this portfolio on GitHub Pages:

1) Create a new GitHub repository.
2) Upload all files inside this folder.
3) Go to Settings > Pages.
4) Choose: Deploy from a branch.
5) Branch: main
6) Folder: / (root)
7) Save.

Your website will work directly without npm or build.

To edit content:
- Open index.html for text.
- Open assets/style.css for colors and design.
- Replace social links in index.html.
- Replace placeholder sections with your real projects.
