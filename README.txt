UPLOAD TO GITHUB PAGES

1) Upload all files directly to your repo root
2) Settings > Pages > Deploy from a branch > main > /(root)
3) Open index.html via your GitHub Pages URL

REAL ONLINE ADMIN (SUPABASE)

1) Create a Supabase project
2) Run supabase-schema.sql in SQL editor
3) Put your Supabase URL and Anon Key in config.js
4) Open admin.html
5) Login password: 22446688
6) Save changes online

Note:
- To upload images easily, create a public storage bucket in Supabase
- Upload image files there and paste their public URL in admin.html
